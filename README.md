# Sistema de Gestión Bancaria

Este proyecto parte de un ejercicio académico desarrollado originalmente en marzo de 2023 para el módulo de **Programación** en Java. 

El objetivo principal es profundizar en la **utilización avanzada de clases**, herencia, interfaces y polimorfismo en Python.

Actualmente, el proyecto se encuentra en una fase de **desarrollo** posteriormente se mrefactorizara para mejorar su arquitectura, limpieza de código y añadir nuevas funcionalidades.

---

## Descripción Original
La aplicación permite la gestión de cuentas bancarias en un banco mediante una interfaz de consola. Utiliza un menú interactivo para realizar operaciones sobre un array de cuentas.
